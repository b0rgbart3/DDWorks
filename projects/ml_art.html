MichaelLawrenceArt.com

This is a custom built artist's portfolio website. Michael Lawrence is a surrealist painter who lives in San Francisco.

Features

  • homepage features a full size image slideshow
  • catalog of images
  • contact form
  • admin panel for uploading new work
  • admin panel for editing the details about each piece
  • admin panel for rearranging the order of the pieces

Technologies

  • HTML5
  • CSS3
  • Javascript
  • PHP
