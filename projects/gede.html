GedeParma.com

This is a custom WordPress theme built for Gede Parma. Gede is an author, teacher, and international speaker.

  • second level page layout has featured images
  • unique second level page layout for books
  • social media links
  • contact form
  • custom logotype and emblem design

Technologies

  • WordPress® and PHP
  • HTML5, CSS3 and Javascript
  • custom theme
