Starburst

A fun simple visual interactive board game for the Apple iPhone® and iPad®

Features

  • simple gameplay
  • highscore tracking
  • option for changing the size of the board
  • slider for changing the color of the board
  • option for solving the puzzle

Technology

  • Written in Objective C++
  • Developed in XCode on the Mac
