LTCCCSF.org

The Long Term Care Coordinating Committee of San Francisco is a governmental agency who's mission is to assist the govenor and other city wide organizations to coordinate efforts related to senior care.

Features

  • custom wordpress theme design
  • mobile friendly smart layout
  • 3 tier navigational menu system
  • accessibility features
  • archive of agendas and meetings notes as pdf documents
  • calendar of upcoming meetings
  • contact form / newsletter subscription list
  • multi-lingual help hotline message

Technologies

  • WordPress®
  • HTML5, CSS3, Javascript & PHP
  • custom plugin for meetings calendar
  • custom plugin for agendas and meetings notes
  • Google Translate (used statically)
