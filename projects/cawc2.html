CaliforniaWitchcamp.org

This was a redesign, the goal of which was to simplify the layout for editing purposes and to add online registration for the event.

  • two-tier navigational menu
  • photo gallery
  • online registration
  • administration dashboard
  • dynamic admin announcements
  • dynamic email lists

Technologies

  • HTML, CSS & PHP
  • PHP and Javascript driven photo gallery
  • MySQL database
  • form processing and admin dashboard written in PHP
