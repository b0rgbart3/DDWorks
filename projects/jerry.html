JerryFrost.com

This is a custom built artist's portfolio website. Jerry Frost is an abstract painter based in San Francisco.

Features

  • image categories
  • full catalog of over 300 artworks
  • admin panel for uploading artwork and show announcements

Technologies

  • PHP, HTML5, CSS3 and Javascript
  • custom php driven admin section for image uploading and creating show announcements
  • MySQL database for storing artwork and show info
