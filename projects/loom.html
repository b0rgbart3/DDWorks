ReclaimingLoom.org

This is a full scale web application that is an online university for metaphysical studies.

Features

  • dynamic content management tool
  • online registration
  • teachers can administer courses and interact with students

Technologies

  • Server API built in NODE JS
  • Front End built in Angular 2
  • Mongo DB hosted on MLab
  • Content storage on AWS
